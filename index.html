Capabilities

Powerful AI Features

Every component is designed for maximum accuracy and broadcast-quality output

Video Analysis

Frame-by-frame analysis using YOLOv8 for precise player and ball detection at every moment of the match.

Multi-Object Tracking

ByteTrack-based IOU matching assigns persistent IDs to every player and the ball across frames.

AI Event Classification

GPT-4o Vision identifies 13 distinct match events — goals, passes, tackles, fouls, and more.

Natural Commentary

GPT generates broadcast-quality play-by-play narration with dynamic pacing matched to video timing.

Text-to-Speech

OpenAI TTS converts commentary into lifelike speech with configurable voice and speed settings.

Video Export

MoviePy composites the original video with AI commentary audio for broadcast-ready output.

Installation

Quick Setup

Get up and running in under 5 minutes

1

Clone & Install

Terminal 
git clone https://github.com/bhatticoder/Automated-Fifa-Commentary.git
cd Automated-Fifa-Commentary
python -m venv .venv && .venv\Scripts\activate
pip install -r requirements.txt
2

Configure API Key

.env 
cp .env.example .env
# Edit .env and add your OpenAI API key
OPENAI_API_KEY=your_openai_api_key_here
3

Run Commentary

Terminal 
python run_standalone.py "path/to/match_video.mp4"
